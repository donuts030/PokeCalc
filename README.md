# Pokemon Damage Calculator

## Project Description
A damage calculator for pokemon, specifically generation 1 and generation 2.

## Technical Description
- Language: Javascript
- Libraries/Framework: React.js
- Editor: VSCode
- API: https://pokeapi.co/
